# Ejercicio con PGP
* https://support.symantec.com/us/en/article.howto41935.html
```CMD
pgp --gen-key "Joe User" --key-type RSA --bits 2048 --passphrase "my passphrase"
pgp --list-keys
pgp --list-keys
pgp --export 0x12345678
pgp --export "Joe User"
pgp --import "Joe User.asc"
```
